# AI for Retail & Commercial

A full-stack demo app showcasing AI-driven decision intelligence for retail, commerce, and marketplace operations. The experience combines scenario simulation, KPI dashboards, and an insights feed to help teams act on demand shifts, pricing pressure, and supply risk.

## Highlights
- Scenario Lab with demand, pricing, and supply simulations
- Executive KPI dashboard for revenue, promo ROI, and stockout risk
- Static insights feed for rapid prototyping on GitHub Pages
- Clean, executive-ready UI for retail decision workflows

## Tech Stack
- Next.js (App Router)
- TypeScript
- Custom CSS
